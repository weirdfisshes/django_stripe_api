Добро пожаловать!

1. Создайте суперпользователя

2. В админке создайте несколько товаров

3. В Перейдите по ссылке для просмотра товара по id, например: http://127.0.0.1:8000/item/1/

4. Нажмите "Buy" и вы будете перенаправлены на Stripe для оплаты

5. После оплаты вы увидите либо подтверждение, что платеж прошел, либо сообщение о невозможности произвести платеж

6. Проект на GitHub: github.com/weirdfisshes/django_stripe_api
