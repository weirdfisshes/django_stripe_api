Добро пожаловать!

1. Убедитесь, что выполнили все шаги из README.md для развертывания проекта (Проект на GitHub: github.com/weirdfisshes/django_stripe_api

)

2. Создайте суперпользователя

3. В админке создайте несколько товаров

4. В Перейдите по ссылке для просмотра товара по id, например: http://127.0.0.1:8000/item/1/

5. Нажмите "Buy" и вы будете перенаправлены на Stripe для оплаты

6. После оплаты вы увидите либо подтверждение, что платеж прошел, либо сообщение о невозможности произвести платеж
